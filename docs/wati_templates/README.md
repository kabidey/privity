# PRIVITY WhatsApp Templates for Wati Approval

## How to Submit Templates to Wati

1. Login to your Wati Dashboard: https://app.wati.io
2. Go to **Message Templates** > **Create Template**
3. For each template below:
   - Copy the **Template Name** (use exactly as shown - lowercase, underscores)
   - Select the **Category** as specified
   - Copy the **Template Body** 
   - Add **Header** if specified (optional)
   - Add **Footer** if specified (optional)
   - Submit for approval

## Template Approval Time
- WhatsApp typically approves templates within 24-48 hours
- Marketing templates may take longer
- Utility/Transaction templates are usually faster

## Important Notes
- Placeholders use format: {{1}}, {{2}}, {{3}}, etc.
- Keep templates professional and within WhatsApp guidelines
- Avoid promotional language in Utility templates
- After approval, the template name in Wati must match exactly what's in our system

## After Approval

Once templates are approved in Wati, update the template mapping in PRIVITY:
1. Go to WhatsApp Settings in PRIVITY
2. Map each system notification to the corresponding Wati template name
3. Test by sending a test message
