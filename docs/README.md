## üì∏ Adding Screenshots

To add screenshots to the tutorial:

1. **Capture Screenshots:**
   - Login to PRIVITY
   - Navigate to each page
   - Use `Windows + Shift + S` (Windows) or `Cmd + Shift + 4` (Mac)
   - Save with these names:
     - `01_login.png`
     - `02_dashboard.png`
     - `03_clients.png`
     - `04_bookings.png`
     - `05_inventory.png`
     - `06_reports.png`

2. **Add to Document:**
   - Place screenshots in `screenshots/` folder
   - Update HTML file: replace placeholder divs with:
     ```html
     <img src="screenshots/01_login.png" alt="Login Page" class="screenshot">
     ```
